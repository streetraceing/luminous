# Luminous

Lightweight dynamic Spicetify theme, idk for what i do this<br>
Youtube video: https://youtu.be/S-2u6xTFZCs

![Luminous image preview](./assets/preview.png)

## Build

### Requirements

- NodeJS 25.5.0
- NPM 11.12.1
- Spicetify 2.43.1

### Local Install & Apply

```bash
npm install
npm run apply
```

### Local Build

```bash
npm install
npm run build
```

## Revert to default theme

```bash
npm run discard
```
